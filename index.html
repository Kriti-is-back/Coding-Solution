In programming, very often we need to ask the computer to check conditional statements. Checking those statements our machine takes decisions. Conditional statements play a great role in programming and about all computer programmers have surely used these things. To write conditional statements we use if statements in programs. 'if' executes the specified statement if the condition is true.

Logical Operators:

MATLAB offers two types of logical operators. One of them operates on every element of an array or matrix( Element-wise ) and the other operates on scalar and logical operations( Short-circuit ). The operators and their uses within if statement are discussed below with examples:

1. Short-circuit operator AND(&&) :

Syntax:

expression1 && expression2

Description:

With logical short-circuiting, the second operand, expression2, is evaluated only when the result is not fully determined by the first operand, expression1. For logical AND (&&) operation, the expression2 will not be evaluated if expression1 is false(0). Each expression must evaluate to a logical result i.e false or true.

Return value for several cases:

Example of AND ( && ) operator:

[tabby title="Matlab"][sourcecode language="matlab"]% Assigning logical values to % expression1 and expression2. expression1 = true; expression2 = false; % ============================================ % Return value of expression1 && expression1: % true && true -> true(1) disp(expression1 && expression1); %============================================== % Return value of expression1 && expression1: % true && false -> false(0) disp(expression1 && expression2); %============================================== % Return value of expression2 && expression1: % false && true -> false(0) disp(expression2 && expression1); %============================================== % Return value of expression2 && expression2: % false && flase -> false(0) disp(expression2 && expression2);[/sourcecode][tabbyending]

Output:

>> gfg1

1

0

0

0

Syntax of AND within if statement:

if( expression1 && expression2) statement(s) end

Note: The above 'if' statement executes if and only if expression1 and expression2 both are true. If one of them becomes false, the 'if' statement does not execute.

Example:

[tabby title="Matlab"][sourcecode language="matlab"]a = 3; % assigning a value to a b = 4; % assigning a value to b if((a >= 3) && (b == 4)) %both conditions are true. disp('Working of short-circuit AND operator.'); end if(a < 3 && b == 4) %first condition is false. % So the second condition is not evaluated % and it returns logical value -> false. disp('GFG'); else % this else part will execute % as the if statement fails to execute. disp('Geeksforgeeks'); end[/sourcecode][tabbyending]

In the above example the first if statement executes because the two expressions inside the if statement are both true. But for the second case, expression1 i.e a < 3 is not true so it does not executes.

Output:

>> gfg

Working of short-circuit AND operator.

Geeksforgeeks

2. Short-circuit operator or( || ) :

Syntax:

expression1 || expression2

Description:

With logical short-circuiting, the second operand, expression2, is evaluated only when the result is not fully determined by the first operand, expression1. For logical OR(||) operation, the expression2 will not be evaluated if expression1 is true(1). Each expression must evaluate to a logical result i.e false or true.

Return value for several cases:

Example of OR( || ) operator:

[tabby title="Matlab"][sourcecode language="matlab"]% Assigning logical value to % expression1 and expression2. expression1 = true; expression2 = false; % ============================================ % Return value of expression1 || expression1: % true || true -> true(1) disp(expression1 || expression1); %============================================== % Return value of expression1 || expression1: % true || false -> true(1) disp(expression1 || expression2); %============================================== % Return value of expression2 || expression1: % false || true -> true(1) disp(expression2 || expression1); %============================================== % Return value of expression2 || expression2: % false || flase -> false(0) disp(expression2 || expression2);[/sourcecode][tabbyending]

>> gfg1

Syntax:

if( expression1 || expression2) statement(s) end

Note: The above 'if' statement executes if either of the expressions (expression1 or expression2) is true. If both of them become false, the 'if' statement does not execute.

Example:

[tabby title="Matlab"][sourcecode language="matlab"]a = 3; % Assigning values to b = 4; % a and b. if((a >= 3) || (b == 4)) % both conditions -> true % So the if statement will execute. disp('Working of short-circuit OR operator.'); end if(a < 3 || b == 4) % first condition -> false % Second condition -> true and returns -> true % the if statement will execute. disp('GFG'); else disp('Geeksforgeeks'); end if(a < 3 || b > 4) % both conditions -> false % So the if statement will not execute. disp('It is not a matlab program.'); else % This else part will execute. disp('Both conditions are false inside if.'); end[/sourcecode][tabbyending]

For the first 'if' statement both the conditions are true and for the second, one of the expressions(first expression) is true but for the last one both of the expressions are false. So, the first and second 'if' statement executes but the last one failed to execute.

3. Logical operations AND, OR, NOT, XOR( &, |, ~, xor ):

Syntax:

if(value1 (any logical operation) value2) statement(s)

end

Note: The above 'if' statement executes if the logical operation of value1 and value2 returns anything other than false and '0' and otherwise the statement does not execute.

  • AND( & ):

Syntax:

if(value1 & value2) statement(s)

end

Note: True /1 -> when value1 & value2 are both true or 1.

False/ 0 -> when either of value1 and value 2 or both are false(0).

Example:

4. all() , any() and islogical() inside if statement:

  • all() : This function determines if all array elements are nonzero or true. It returns true if all array elements are nonzero and returns false otherwise.

Example:

[tabby title="Matlab"][sourcecode language="matlab"]a = [1; 1; 2; -1]; b = [0; 0; 0]; if(all(a)) % returns -> true disp('all array elements are nonzero.') else disp("Not all array elements are nonzero.") end if(all(b)) % returns -> false disp('all array elements are nonzero.') else disp("Not all array elements are nonzero.") end[/sourcecode][tabbyending]

Output:

>> gfg

all array elements are nonzero.

Not all array elements are nonzero.

  • any() : This function returns true if any of the array elements are nonzero and otherwise returns false.

Example:

[tabby title="Matlab"][sourcecode language="matlab"]a = [1; 1; 2; -1]; b = [0; 0; 1]; if(any(a)) % returns -> true disp('any of array elements are nonzero.') else disp("Not any array elements are nonzero.") end if(any(b)) % returns -> true disp('any of array elements are nonzero.') else disp("Not any array elements are nonzero.") end[/sourcecode][tabbyending]

Output:

>> gfg

all array elements are nonzero.

all array elements are nonzero.

  • islogical(): islogical(A) returns true if A is a logical array and false otherwise. islogical also returns true if A is an instance of a class that is derived from the logical class.

Examples:

[tabby title="Matlab"][sourcecode language="matlab"]a = [1; 1; 2; -1]; b = [false; false; true]; if(islogical(a)) % returns -> false disp('all array elements are logical.'); else disp('Not all array elements are logical.'); end if(islogical(b)) % returns -> true disp('all array elements are logical.'); else disp('Not all array elements are logical.'); end if(islogical(logical(a))) %returns -> true disp('all array elements are logical.'); else disp('Not all array elements are logical.'); end[/sourcecode][tabbyending]
